# bt_api_bitmart

BitMart exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_bitmart
```

## Usage

```python
from bt_api_bitmart import BitmartExchangeDataSpot, BitmartRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA256 authentication with memo
- Symbol format: `BTC/USDT` -> `BTC_USDT`
